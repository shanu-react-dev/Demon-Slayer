## Colors in CSS

_Color is a property in css to add the color for the elements or its background._
_There are several ways to add the colors to the elements._

1. **Named Colors:** By using this we can add the color by its name simply target the elements use color property and add the color name as value.
2. **RGB Function:** This function accepts 3 values and each value should be between 0 to 255. This is combination of Red, green and blue color.
3. **RGBA Function:** Similar to RGB but with additional feature alpha which represents the degree of darkness or lightness of a color. Alpha value lies between 0 to 1 or we can go with percentage as well.
4. **Hex Codes:** The hex code value is an optimized way to add the colors for our elements. Here it is combination of characters `a to f` and digits `0-9`. To write hex codes color we need to use special symbol `#` as a prefix followed by combination of hex codes.
   _Hex code follows pairing of the colors magnitude as RGB `#123` here value 1 is for red color 2 for green color and 3 for blue color. we can go with 6 characters also here each 2 digit will be having pair as red green blue._
5. **HSL function:** This function contains the combination of `Hue` which represents color wheel from 0 to 360 and `S` represents saturation which is brightness of the color and `L` is for lightness of the colors. we can use percentage value for saturation and lightness from 0 to 100.
