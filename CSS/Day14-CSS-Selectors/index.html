CSS Selectors

  1. Simple Selector
    1. ID
      ID selector is a simple selector and it selects the elements based on their id attribute value. It uses # symbol as a prefix to the attribute value in CSS. It will target the entire elements having same id in the HTML file.
      ID is unique in nature. Id should have only one value not multiple
    2. Class
      Class Selector is a simple selector only and it is used to target the elements based on their class names. There may be multiple values inside class attribute. We need to use dot symbol as a prefix to target the elements based on their class name. It will target the entire elements having same class name in the HTML file.
    3. Tagname
      Tagname falls under simple selector, It helps to target the elements based on their tag names. It will select the entire element having same name in the file.
    4. Grouping
      Grouping selector comes under Simple selector and it helps to target the multiple elements at the same time by separating with comma (,). It will select the entire element whichever has been targeted in the HTML file.
    5. Universal
      This selector targets the entire element inside an HTML file. The styles properties written inside universal selector will be applicable for all elements. Using * symbol we can use universal selector .
