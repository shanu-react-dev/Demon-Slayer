## Fonts and Text in CSS

_In CSS, the terms font and text refer to two different categories of styling properties that affect how words appear on a screen._

## Differences

- **Font properties (font)** define the shape and design of the individual characters (glyphs). They deal with things typically found inside a font file, like weight, style, and size.

- **Text properties (text)** define the layout and decoration of those characters within a container. They handle how the characters are aligned, spaced, or decorated (like underlining), which are not part of the font file itself.
