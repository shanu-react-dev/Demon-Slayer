Combinator Selector

  1. Descendent Selector
    It is a combinator selector used to target the direct as well as indirect child of an element. To use this selector first we need to write parent element then the elements needs to target with space.
    e.g. ul li properties:value
    This will target the all list items used inside the unordered list whether it is ordered list items as well.
  2. Child Selector
    This selector falls under Combinator selector and it targets all direct child elements of a parent element. To use this selector we need to use greater than symbol > after parent element then the elements needs to target.
    e.g. ul>li
    In the above syntax it will target only the list items which is present inside the unordered list.
  3. Adjacent Sibling Selector
    This selector comes under Combinator selector and used to select or target the nearest sibling elements. To use this we need to use plus symbol.
    e.g.: h1 + h2 {properties: value} In the above code it will target only the h2 element whihc is present just after h1.
  4. General Sibling Selector
    It also comes under combinator selector and it targets the entire sibling elements not only the nearest sibling of an element. To use this we need to use tilde symbol.
    e.g.: h1 ~ h3 {properties: value;} Here the entire h3 sibling will be selected which is a sibling of h1.
