# Formatting tags
_Formatting tags in HTML are elements used to style or emphasize text—either visually (like bold/italic) or semantically (giving meaning, which helps accessibility and SEO). Few examples are as follows._

- `<b></b>`: This is known as bold tag generally used to make the content as bold. It does not have any importance.
- `<strong>`: This is strong tag and workwise it is similar to bold tag but it adds some extra importance to the text. It improves SEO by adding semantics.
- `<i>`: This is known as italic tag generally used to make the content as italicize. It does not provide any importance to the text or content.
- `<em>`: This tag is known as Emphasize tag and it is used to make the text italic only but with semantics so it will help to improve the SEO as well.
- `<u>`: This tag is known as underline tag and it is used to underline the text but it does not add any importance to the text.
- `<ins>`: This is also known as insert tag it adds semantics and underlines the text.
- `<pre>`: This is known as Pre-Formatted text and used to show the content as it is on the UI the way you write in HTML file.
- `<mark>`: This is a semantic tag and used to highlight the text. 
- `<sup>`: This is also known as SuperScript and used to write mathematical formulas.
- `<sub>`: THis is also known as SubScript and generally used to write the chemical formula.
- `<del>`: This is known as deleted text and it will show the content as striked.
- `<s>`: It behaves like `<del>` tag but it does not have any importance.
- `<code>`: It is used to embed any kind of coding text related to any programming language also we can addd. It simply change the font style.
