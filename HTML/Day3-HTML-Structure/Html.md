## HTML

_HTML stands for Hyper Text Markup Language. It is a standard Markup language used to create structure of the webpages. It is also used to add data to the webpages such as Text, Images, videos, gifs etc.._

## Hyper text

_Hyper Text refers to a text which contains some extra information. In the context of HTML tags can be considered as HyperText._

## Markup Language

_A markup langugage generally refers to creating a structure of the websites. In HTML we have predefined tags by which we can create structures of the websites or webpages._

## Structure of HTML

```html
<!-- Version of HTML or showing we are using HTML document -->
<!DOCTYPE html>
<!-- Root Element -->
<html lang="en">
  <!-- Head tag contains the meta data or whatever the things is written inside head tag is totally for browser understanding -->
  <head>
    <!-- Meta tag contains metadata -->
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width initial-scale=1.0" />
    <!-- Title represents the name of the webpages -->
    <title>Document</title>
  </head>
  <!-- Body tag contains the data which will be visible to our users. -->
  <body>
    body tag contains the data which can be visible to users.
  </body>
</html>
```

## History of HTML

_HTML was introduced in 1991, by Tim Berners Lee. The main intent was behind introducing HTML is to show the data in structured Format. The first version of HTML was introduced in 1991 itself. Currently we are using HTML5 as per 2026._
