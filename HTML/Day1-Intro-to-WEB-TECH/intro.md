## Web Technology
*Web Technology is a Technology which helps us to create Websites or Web Apps. It includes Frontend, Backend, Databases.*

## Frontend
*Frontend is known as UI or User Interface. This is a part of web technologies which is responsible for viewing the interfaces for any applications or websites. We can build frontend using technologies like HTML, CSS and JavaScript.*
## Backend
*Backend is also known as Middleware it contains all logical codes. Here generally we are having IP address for the websites.*

## Database
*Database is a storage where we are storing multiple data in the organised or systematic manner. It uses technologies like SQL, MongoDB, Cloud Storage (AWS, Azure) etc.*

## Websites and Web Apps
*Websites and Web Apps both are collection of multiple webpages. Generally Websites are static in nature and Web Apps are dynamic in nature. A website contains multiple data such as Images, videos, text etc. It provides a global representation of any information or data.*
