## Tables in HTML

_Table is a combination of rows and columns. It is used to show the data in structured format. We can create tables using `<table>` tag in HTML._

## `<caption>` Tag

_Caption tag is generally used inside table tag to give the name to the tables. Which aligns the content in center respect to the table._

## `<thead>`, `<tbody>` & `<tfoot>`

_These are the tags which holds the table content. `<thead>` tag stores the table's column or row heading. Using `<th>` tag we can create headings for our columns._

_`<tbody>` tag stores the actual data for the tables. Inside this we have `<tr>` tag to create rows and to add data in the rows we use `<td>` tag. `<td>` tag also known as cells._

_`<tfoot>` tag represents the summarized data of tables. Inside this also we can use `<tr>` tag and `<th>` tag to make the text bold and in center of the table cells._

## Rowspan and Colspan

_Rowspan and Colspan both are attributes in HTML tables generally used to merge two or more rowcells or column cells. To merge two or more column cells we use colspan attribute it accepts value or count of the cells we need to merge. Likewise rowspan is used to merge two or more row cells._
