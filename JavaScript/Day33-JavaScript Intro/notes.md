## JavaScript

_JavaScript is a High Level Programming language as well as Client Side Scripting Language. It is used to add functionalities to static or any webpages._

_It was introduced in 1995 by **Brendan Eich** in just only 10 days. First it was introduced as **Mocha**. Later it got renamed as **LiveScript** so that it could match to its work. Later again it got renamed as **JavaScript** just for marketing purpose._

_JavaScript got one competitor in 1996 as Jscript which was introduced by Microsoft. To overcome up with confusion which one is better ECMA (European Computer Manufacturer Association) came in the picture. It found JavaScript bit more optimised as compared to Jscript._

_JavaScript again got renamed as EcmaScript after that ECMA only providing the updates for our JavaScript. Currently we are using **ES16** version of JavaScript._

_JavaScript can run on server side using **NodeJS** and in frontend we don't need any extra software to run JavaScript. Browser is more than enough._

_NodeJS is just a JavaScript Runtime Environment and it allows JavaScript to run on the server side._

## Features of JavaScript

1. **High Level Programming Language:** Easily understandable by humans.
2. **CLient Side Scriptign Language:** Without any server it can perform changes to the UI or No need of server to run the code.
3. **Synchronous in Nature:** Only one task can be done at one time.
4. **Interpreted:** It executes the code line by line. Until the first line has been executed it will not go for next line.
5. **Object Based:** Each and everything is considered as an object here.
6. **Dynamically typed:** In JavaScript the datatype will be checked at runtime or No need to specify the datatype while declaring the variables.
