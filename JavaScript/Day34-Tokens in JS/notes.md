## Tokens

_Tokens are th smallest unit or part of any programming language. It has totally 5 types in JavaScript._

1. **Keywords:** These are predefined words having some special meaning known as keywords. There are multiple keywords such as `var`, `let`, `const`, `new` etc..
2. **Identifiers:** Identifiers are the name of created variables, functions or classes. It has certain rules as follows.
   1. It must and should not start with numbers or any special characters except underscore `_` or dollar `$`
   2. It can include numbers in between but not from starting.
   3. There should not be any white space in between identifiers.
   4. Keywords are not allowed as an identifier.
3. **Literals:** These are the actual data assigned to a variable. There are two types of literals in JavaScript.
   1. **Primitive Literals:** These are single value datatype. It is value based and it is immutable. It can't be changed after assigning.
   2. **Non-Primitive Literals:** These are multiple value datatype. It is reference based and it is mutable as well. It can be modified or changed.
4. **Separators:** These are special symols provide separation of concerns to the codes. There are multiple separators it can be a block `{}`, `;` etc.
5. **Operators:** Operators are predefined symbol used to perform some specific tasks based on operands or variables. There are multiple operators such as Arithmetic, Assignment, Logical, Relationals, Unary, Ternary operator etc...

<p align="center"><b>Primitive DataTypes in JavaScript</b></p>

_There are total 7 primitive datatypes in JavaScript._

1. **Number:** It represents any numerical value whether it is integer value, float, double or byte.
2. **String:** In JavaScript string is a primitive datatype. It is a sequence of multiple characters enclosed within either double quotes `"abc"` or single quotes `'abc'`.
3. **Boolean:** This primitive datatype consists of two values either `true` or `false`. It is used to check conditions between variables.
4. **Null:** It is a representation of empty object. We can say here null is an empty value but it is not zero.

   _The type of null is always an object_

5. **Undefined:** Undefined is something which is not defined. It is a default value for an unassigned variable. If any variable is not initialized with a value then undefined will be the default value.
6. **Bigint:** This can accept any integer value which is greater than 16 digit.
7. **Symbol:** It is a primitive datatype used to make the specific variable as unique. It can be used while writing logical programmes.
